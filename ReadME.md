## 👋 Sobre este Repositorio
Este espacio está diseñado para estudiantes creativos que están aprendiendo a traducir su visión gráfica al lenguaje de la web. Aquí encontrarás el código fuente de las clases, los "Code Challenges" semanales y las soluciones a los ejercicios de laboratorio.

**Objetivo:** Dejar de ver la programación como matemáticas aburridas y empezar a verla como una herramienta de diseño más potente que cualquier software de edición.

---

## 📂 Estructura del Curso

El contenido está organizado por semanas y unidades temáticas, siguiendo el paradigma de **aprendizaje experiencial** (Aprender haciendo).

### 🏛️ Unidad I: Infraestructura y Estructura (Semana 1)
*Entendiendo la arquitectura de la web y el esqueleto semántico.*
- [`/semana-01/autopsia-web`](./semana-01) - **Reto:** Hackeando el DOM de un sitio de noticias.
- [`/semana-01/mi-presentacion`](./semana-01) - **Práctica:** Estructura HTML semántica (Header, Main, Footer).

### 🎨 Unidad II: Estilizando la Web (Semana 2)
*Dominando el CSS, Box Model y el posicionamiento.*
- [`/semana-02/polaroid-challenge`](./semana-02) - **Ejercicio:** Box Model y Sombras en tiempo real.
- [`/semana-02/bauhaus-poster`](./semana-02) - **Reto Creativo:** Recreando pósters de diseño Suizo con puro CSS (Sin imágenes).

### ⚡ Unidad III: Interactividad y DOM (Semana 3 - 4)
*Dando vida al diseño con JavaScript.*
- [`/semana-03/consola-logica`](./semana-03) - Variables y tipos de datos (El perfil de usuario).
- [`/semana-03/titiritero-dom`](./semana-03) - **Práctica:** Manipulación del DOM y Eventos (Dark Mode & Mood Board).
- [`/semana-04/landing-page`](./semana-04) - **Proyecto Parcial 1:** Portafolio personal interactivo.

### 🚀 Unidad IV: Frameworks y Futuro (Próximamente)
*Introducción a Vue.js y consumo de APIs.*

Para seguir este curso, asegúrate de tener instalado:

3.  **[Git](https://git-scm.com/)**: Para el control de versiones.

## 💻 Cómo usar este repo

Si eres alumno, puedes descargar el código para estudiarlo:

1.  Dale click al botón verde **Code**.
2.  Selecciona **Download ZIP**.
3.  Descomprime y abre la carpeta de la semana correspondiente en VS Code.

O si ya te sientes hacker, usa la terminal:
```bash
git clone [https://github.com/TuUsuario/programacion-sitios-dinamicos.git](https://github.com/TuUsuario/programacion-sitios-dinamicos.git)

```
